# Fonts

Self-hosted web fonts for the AEGIS design system.

## Required Font Files

Copy these from the aegis-constitution repository's `public/fonts/` directory:

- `ibmplexsans-variablefont_wdthwght-webfont.woff2` — IBM Plex Sans (variable, normal)
- `ibmplexsans-italic-variablefont_wdthwght-webfont.woff2` — IBM Plex Sans (variable, italic)
- `poppins-light-webfont.woff2` — Poppins Light (300)
- `poppins-regular-webfont.woff2` — Poppins Regular (400)
- `poppins-medium-webfont.woff2` — Poppins Medium (500)
- `poppins-semibold-webfont.woff2` — Poppins SemiBold (600)

## Why self-hosted?

Performance and privacy. No external font CDN requests. Fonts are preloaded
in the HTML head for optimal loading.
