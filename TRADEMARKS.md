## Permitted Uses

The following uses are permitted without explicit permission:

### Reference and Citation

- Referencing the AEGIS™ architecture in technical documentation
- Academic papers, research materials, and educational content
- News articles and blog posts discussing the AEGIS™ project
- Implementation documentation for AEGIS-compatible systems

### Conference Talks and Presentations

When using AEGIS™ in presentation titles or materials:

**Recommended formats:**

- "Implementing Architectural Enforcement & Governance of Intelligent Systems (AEGIS™)"
- "Building AI Safety with AEGIS™ Governance"
- "AEGIS™: A New Approach to AI Governance"

**Best practices:**

- Use the full name on first reference when space permits
- Include the ™ symbol on first mention in title or abstract
- Clearly indicate the talk references this project, not a generic system

### Implementation Projects

Projects implementing the AEGIS specifications may:

- State compatibility: "Compatible with AEGIS™ specifications"
- Reference the architecture: "Built using AEGIS™ governance patterns"
- Link to this repository for context

**Note:** Implementation projects should use their own product names and branding, not appropriate the AEGIS™ mark as their primary identifier.

## Trademark Symbol Usage

### When to use ™

- **First reference** in any document, presentation, or material
- Product names, titles, and prominent headings
- Marketing and promotional materials

### When ™ is optional

- Subsequent mentions in the same document after first use
- Informal discussions or casual references
- Source code comments (though encouraged for clarity)

### The "AEGIS" name without ™

Use of "AEGIS" without the ™ symbol in descriptive or referential contexts (e.g., "the AEGIS architecture", "AEGIS governance model") does not require permission, provided:

- The use is clearly referencing this project
- The use does not imply trademark ownership
- The use does not suggest official endorsement

However, **best practice is to include the ™ symbol** on first mention to maintain proper trademark notice.
