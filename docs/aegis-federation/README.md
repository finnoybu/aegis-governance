# AEGIS Governance Network Pack

This folder contains the initial documentation set for the AEGIS Governance Federation Network (GFN):

- AEGIS_ATPROTO_GOVERNANCE_NETWORK.md
- AEGIS_ATPROTO_SCHEMA.md
- AEGIS_NODE_REFERENCE_ARCHITECTURE.md
- AEGIS_GOVERNANCE_FEEDS.md
- AEGIS_TRUST_MODEL.md

The federation network enables distributed sharing of governance intelligence.

Nodes publish and subscribe to governance signals such as:

- policy updates
- circumvention techniques
- risk alerts
- governance attestations
