# Code of Conduct

## Our Commitment

We as contributors and maintainers pledge to make participation in the AEGIS™ project a respectful and harassment-free experience for everyone.

We are committed to fostering an open, collaborative environment focused on advancing responsible AI governance.

---

## Expected Behavior

Participants are expected to:

- Be respectful and considerate in discussions
- Focus on constructive technical dialogue
- Provide helpful feedback and thoughtful critiques
- Respect differing viewpoints and experiences

---

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or discrimination
- Personal attacks or insulting language
- Publishing private information without consent
- Disruptive or intentionally inflammatory conduct

---

## Enforcement

Project maintainers may remove, edit, or reject contributions or comments that violate this Code of Conduct.

Repeated violations may result in temporary or permanent removal from project participation.

---

## Scope

This Code of Conduct applies to:

- GitHub issues
- pull requests
- discussions
- project communication channels

---

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers.

Reports will be reviewed and addressed promptly and confidentially.

---

This Code of Conduct is adapted from the **Contributor Covenant**.
